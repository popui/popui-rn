单选框

## API

### Radio

属性 | 说明 | 类型 | 默认值
----|-----|------|------
| name    |   name  | String |   无  |
| defaultChecked |   初始是否选中   | Boolean  | 无  |
| checked    |   指定当前是否选中  | Boolean  | 无  |
| disabled      |  禁用  | Boolean |  false  |
| onChange    | change 事件触发的回调函数 | (e: Object): void |   无  |
