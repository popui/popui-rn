`popui-rn` 是移动规范的 React Native 实现,融合了 Weui,Ant Design 等其他 UI。


## 特性

- 规则化的视觉样式配置，适应各类产品风格。
- 基于 React Native 的多平台支持。
- 使用 TypeScript 开发，提供类型定义文件。
